# 📊 Global Electronics Sales Analysis

## 🎯 Overview  
Analyzed a global electronics dataset to understand customer behavior, product performance, sales trends, and store efficiency using SQL and Power BI.

---

## 🧹 Data Preparation  
- Cleaned and standardized datasets (customers, sales, products, stores, exchange rates)  
- Handled missing values and converted data types for accurate analysis  

---

## 🔍 Key Insights  
- USA dominates (~44% customers) → high regional dependency  
- Majority customers are 30+ → low youth engagement  
- Computers category drives highest revenue and profit  
- Desktop PCs are top-selling and highest revenue products  
- Strong seasonality with peaks in Dec–Feb  
- Revenue concentrated among few products and customers  
- Some stores underperform with low sales and profit  

---

## 📊 Dashboard (Power BI)  
- Built interactive dashboard with KPIs and performance insights  
- Used bookmarks for dynamic views (Top/Bottom products, Revenue/Orders/Profit)  

---

## 🚀 Recommendations  
- Expand in underperforming regions (Italy, Australia)  
- Focus on high-margin categories (Computers, Appliances)  
- Optimize or improve low-performing stores  
- Target younger customers with new strategies  
- Increase average order value using bundles and offers  

---

## 🏁 Conclusion  
Converted raw data into actionable insights to support business growth, improve performance, and reduce risks.
